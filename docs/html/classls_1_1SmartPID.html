ls::SmartPID::SmartPID ( double cc = 1,
double w = 10,
double lc = 0.001,
double max = 127 )
explicit

Construct a new SmartPID object.

Attention - This algorithm creates really fast changes in output which can be dangerous to the robot if correct parameters are not provided. change the settings with seperate warnings only if you know what you are doing. More about SPID can be looked up in the LibStoga documentation.

Parameters
ccthe correction constant. determins the speed of the overall motion.
wwindup value for integral. will also determine how much the integral will be dominant in the motion. USE ONLY IF YOU KNOW WHAT YOU ARE DOING.
lcthe learning constant. the rate at which it should learn. CHANGE ONLY IF YOU KNOW WHAT YOU ARE DOING.
maxthe maximum value of the PID output.
