Detailed Description

Complete PID controller gain configuration.

This structure encapsulates all parameters needed to configure a PID controller, including the three main gains (P, I, D) and advanced features like integral windup protection and zero-crossing reset.

The structure provides sensible defaults that work for many applications, but should be tuned for specific use cases.
