Detailed Description

Complete robot pose representation (position + orientation)

Represents the full state of a robot in 2D space including X, Y coordinates and heading angle. This structure is fundamental to odometry and navigation systems throughout the framework.

The coordinate system uses:

  • Origin: Robot's starting position (0, 0)
  • Angles: Bearing system where 0° = North (forward direction)
  • Units: All distances in inches, angles in degrees
